FlashVars Overview

Variable name
Parameters
 Description
wmode
Transparent, Opaque
 This is used to change the apperance of the swf files background.
allowFullScreen
true/false
 Set this to true to enable viewing in fullscreen.
xmlFile
xml file
 The direct path to the xml file. Note: you only need to include this in the FlashVars if you are wanting to change the name of the xml file.
