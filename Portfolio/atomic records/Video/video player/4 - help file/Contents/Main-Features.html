Package content

The package you purchased contains the following features:

  • Completely skinnable interface, all the settings are xml driven, for ease of use.
  • Fullscreen video resizing with smoothing for optimal picture quality.
  • Plays videos in HD h264 format as well as the standard Flash format.
  • Six keyboardEvents that control some of the players functionality, these keyboardEvents are listed below and can be added or removed via the xml file.
  • Auto play, true/false. Set to true plays the video as soon as the page comes up, or set to false displays a preview image.
  • A Watermark with external link can be place anywhere on the screen, the options are: TL, TR, BL, BR.
  • A right click contextMenu, allowing you to display your company details to prospective clients.
  • global.css (used to define html formatting in description) and can be used to change font color, size or to add url links.
  • Playlist which supports unlimited thumbs.

KeyboardEvents

The keyboardEvents are as follows:

  • spaceKey: toggles the playPause button
  • ctrlKey: resizes the video
  • shiftKey: toggles the mute button
  • upArrow: toggles the description panel up/down
  • downArrow: toggles the playlist up/down
  • enterKey: toggles the playPause button for the thumbs slideshow

Note: KeyboardEvents can only be used once you have registed it with a mouse click.
