Overview

The Video Player is both Elegant and Stylish with some thoughtful features. All of the functionalities are optional and can be set according to your own needs. One of the most exciting new feature is the slideshow thumbs which adds a nice touch to the way you deliver your content.

With no Flash knowledge required this video player is easy to setup and simple to use. Included in this help file is the embed code for embbedding the player into your html page as well as a list of all the xml settings you need to help you get started.


The package you purchased contains the following:

  • 1 - Source FLA: original source files, you dont need to upload any of these files to your server. Use these to change the code or edit graphics, Note: a new swf file will be published to the Published SWF - Folder
  • 2 - Published SWF: published swf file, html file, xml folder, js code for embedding the swf file, these are the files you need to upload to your server.
  • 3 - PSD: contains the photoshop document for the logo
  • 4 - Help File: contains the help documents

XML overview

The xml file that comes with this package has a set number of settings. The xml settings allows you to setup the player the way you want it. Below is an overall structure of the xml settings:

Variable name Variable type Value range Description
XML Settings
width Number Width of the video player area in px.
height Number Height of the video player area in px.
textColor Hex-Decimal 0x000000 This is the text color for the video title, this is a hex-decimal number and must be written as shown in the "value range" column.
dropShadow Boolean true/false Set to true creates a dropShadow for the title text.
playerSkin String black/white This video player comes with two skins, black and white, to change the skin just specify which skin you want to use. If left blank the default black skin will be used.
timeText Hex-Decimal 0x000000 This is the text color for the video time text, this is a hex-decimal number and must be written as shown in the "value range" column.
autoPlay Boolean true/false If you want the video player to start playing automatically, then set this to true, otherwise set this to false.
initVolume Number 0 - 1 The default starting volume. Enter any number between 0 and 1. Note: the start volume will be overwritten once the end user clicks on the volume control.
aspectRatio Number 1 - 3 This is the start size of the video, options are as follows 1:Normal size, 2:Stretch, 3:Full size.
listeners String listeners relates to the keyboard events that control some of the players functionality. To learn more about the listeners just go to "Video Player AS3 - Main Features" in the List of Contents.
copyright String Copyright is frequently used on site's to protect intellectual property.
notice String Use this to display information about your site, such as company name or contact us.
link String This is an external link, which will re-direct the user to your site when they click on the notice.
useControls Boolean true/false This refers to the slideshow controls for the thumbs. Set to true the controls will be visible on the screen when the thumbs come up.
hide Boolean true/false This refers to the playlist and is only needed if autoPlay is set to false. if set to true the playlist will not be visible once the player has been loaded in.
hideDelay Number This is the timer for the thumbs slideshow, and is used to skip to the next row of thumbs onTimerComplete.
autoStart Boolean true/false This refers to the thumbs slideshow, set to true starts the timer for the slideshow.
